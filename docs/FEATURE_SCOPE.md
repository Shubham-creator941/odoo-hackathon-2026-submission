# Feature Scope

This document defines the locked scope for the hackathon.

No new features should be introduced without team approval.

---

# Admin Portal

## Dashboard

- Total Employees
- Present Today
- Employees On Leave
- Pending Leave Requests
- Payroll Overview
- Recent Activities

---

## Employee Management

- Employee List
- Search
- Filter
- Add Employee
- Edit Employee
- Delete Employee
- Employee Details

---

## Attendance Management

- Attendance List
- Search
- Filter
- Employee Attendance History

Admin does NOT mark attendance.

---

## Leave Management

- View Leave Requests
- Search
- Filter
- Approve Leave
- Reject Leave
- Admin Comment
- View Leave Details

Admin does NOT apply for leave.

---

## Payroll

- Payroll List
- Salary Details
- Edit Salary
- Salary History

---

## Documents

- Upload
- Assign
- View
- Delete

---

## Notifications

- View Notifications
- Mark as Read
- Delete

Notifications are informational only.

---

## Profile

- View Profile
- Edit Profile
- Change Password

---

- AI Chatbot
